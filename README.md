# CSV Comparer

This python tool allows you to browse your computer for two CSV files, choose a column to compare (columns must have the same name), and run exported reports that show you which values are only in File A and which ones are only in File B.

This is a great tool for comparing large datasets in Excel from database queries without the hassle of highlighting duplicates or working with messy formulas to compare two datasets.

# Use

1. Add your data to 2 CSV files
2. Run the tool and choose your two CSV files
3. Choose a column to compare on from the dropdown
4. Click Compare
5. Tool shows you the comparison in a front end GUI window, but also saves the results to two separate csv files in the 'Results' folder
